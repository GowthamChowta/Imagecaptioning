# Imagecaptioning

Data:
https://www.kaggle.com/shadabhussain/flickr8k

Project Goal:
Give an image, generate an automatic caption for the image.

### Model:
Enoder - Resnet50 with image net weights <br>
Decoder - LSTM and dense layer.
